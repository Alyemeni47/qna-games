# لعبة سؤال وجواب (Q&A Game)

لعبة جماعية تفاعلية تعمل في المتصفح بدون الحاجة لتسجيل دخول، تدعم اللغتين العربية والإنجليزية مع تبديل فوري للاتجاه.

## المميزات

- تصميم حماسي وجذاب
- دعم كامل للغة العربية والإنجليزية (RTL/LTR)
- 6 فئات مختلفة للأسئلة
- 36 سؤال اختيار من متعدد
- إنشاء فرق متعددة
- مؤقت زمني لكل سؤال
- نظام نقاط تفاعلي
- 3 وسائل مساعدة لكل فريق:
  1. حذف خيارين غير صحيحين
  2. تغيير السؤال الحالي
  3. مضاعفة النقاط للإجابة الصحيحة
- شاشة نتائج نهائية
- تصميم متجاوب يعمل على جميع الأجهزة

## كيفية التشغيل

### الخيار 1: النشر على Vercel
1. انشئ حساب على [Vercel](https://vercel.com)
2. انقر على "New Project"
3. اختر مستودع GitHub الذي يحتوي على المشروع
4. اضغط على "Deploy"

### الخيار 2: التشغيل محليًا
1. قم بتنزيل الملفات
2. افتح الملف `index.html` في متصفحك
